GNU/Linux Native

The Oxygen Fonts have been developed 'natively' on GNU/Linux systems, mainly using FontForge (below).

Developing the Oxygen Fonts natively on a GNU+Linux system with FontForge has allowed the fonts to be developed closely with Freetype's render and hinting engines, so that the fonts can be optimised for the pixel grid fitting on Freetype based OS's (below) such as the various GNU/Linux and similar systems, including Chrome OS and Android.

Getting Oxygen Fonts

Download OxygenFonts.zip

The Oxygen fonts can be downloaded as font files, or you can clone and pull the whole development repository from github.
