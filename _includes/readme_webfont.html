Oxygen as a Webfont

An early version of Oxygen Font was released as part of the Google Fonts project.

Although released at an early stage of development with only 3 weights, the Oxygen webfont proved to be a popular font for websites and framework theme designers. This early version of Oxygen is served by Google's Font API over 178,300,000 times a week.

Feedback and usage stats from the Oxygen webfont have been an integral part of shaping the further development and refinement of the Oxygen Font family.
